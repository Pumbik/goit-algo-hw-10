# Обчислення визначеного інтеграла методом Монте-Карло

## Опис завдання
Обчислити значення інтеграла для функції $f(x) = x^2$ на відрізку від 0 до 2.
Простими словами: знайти **площу зафарбованої фігури** під графіком цієї функції (сіра зона на графіку).

Для цього ми використали два способи:
1.  **Метод Монте-Карло** — це метод "випадкових пострілів". Я навмання ставил багато точок на графік і рахувал, який відсоток із них потрапив у потрібну зону (під лінію функції).
2.  **Функція `quad` (з бібліотеки SciPy)** — це стандартний математичний інструмент, який рахує інтеграл дуже точно. Я використал його як еталон, щоб перевірити, наскільки помиляється наш метод Монте-Карло.

## Результати
* **Теоретично точне значення:** $2.66666...$ ($8/3$)
* **Результат функції `quad`:** $2.66666...$ (ідеально точно)
* **Результат методу Монте-Карло:** Приблизно $2.66...$ (результат трохи змінюється при кожному запуску).

## Висновки

1.  **Точність проти Випадковості:**
    Метод Монте-Карло дає **наближений** результат. Оскільки він базується на випадкових числах, відповідь кожного разу трохи різна. Натомість функція `quad` дає **стабільний і точний** результат.

2.  **Роль кількості точок:**
    Точність методу Монте-Карло напряму залежить від кількості "пострілів" (точок). Чим більше точок ми беремо (наприклад, 15 000 або 100 000), тим ближче наш результат до ідеального. Якщо точок мало — похибка буде великою.

3.  **Для чого це потрібно?**
    Для простих задач (як ця) метод Монте-Карло здається менш ефективним, бо він повільніший і менш точний за звичайні формули. Проте його справжня сила розкривається у **складних задачах** (наприклад, у фізиці чи фінансах), де неможливо застосувати звичайні формули. Там "бомбардування" випадковими числами часто стає єдиним способом знайти відповідь.
